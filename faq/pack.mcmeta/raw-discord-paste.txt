--= **Code to go in `pack.mcmeta`** =--
```json
{
    "pack": {
      "pack_format": #,
      "description": "Something Interresting"
    }
}```
• Replace `Something Interresting` with a description
• Replace `#` with an integer,
**Resource Pack**
`1` = 1.6.1 to 1.8.9
`2` = 1.9 to 1.10.2
`3` = 1.11 to 1.12.2
`4` = 1.13 to 1.14.4
`5` = 1.15 to 1.16.1
`6` = 1.16.2
**Datapack**
• Replace `Something Interresting` with a description
• Replace `#` with an integer,
`3` = 1.13 to 1.13.2
`4` = 1.14 to 1.14.4
`5` = 1.15 to 1.15.2
`6` = 1.16.2
