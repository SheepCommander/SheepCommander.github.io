**VSCode**
• Has lot's more plugins for DPs
• Plugins are easier to install
• Has more built-in tools
• Is free, doesn't constantly send you a message to buy the licensed version
• Has integration for GitHub & Git as well as plugins for further integration

**Sublime Text 3**
• Instantly starts up (VSCode can take a bit)
