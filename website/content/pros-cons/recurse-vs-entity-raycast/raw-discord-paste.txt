**Normal**
• Less lag for not using an entity
• Less lag for not having to `tp` an entity
• Less lag for not having to run as many commands
• Less steps, only needs 1 command to recurse
• Less steps, doesn't have to summon an entity
• Less steps, doesn't have to kill the entity

**Entity**
• Can be easier to use for Slow Raycasting
