VSCode Plugins

language-mcfunction

• Adds syntax highlighting (Color) and can also point out errors

https://marketplace.visualstudio.com/items?itemName=arcensoth.language-mcfunction

DataPack Helper Plus

• Autofills commands and other, points out errors

https://marketplace.visualstudio.com/items?itemName=SPGoding.datapack-language-server

JSON .mcmeta

• Allows VSCode to read .mcmeta files as JSON and can

https://marketplace.visualstudio.com/items?itemName=xTeal.mcmeta#review-details

Tellraw Preview

• Shows you a preview of what a tellraw/say/similar command would output (No color), points out errors.

https://marketplace.visualstudio.com/items?itemName=actuallyboomber.tellraw-preview

MC DataPack

• Allows you to easily generate DPs that follow the DP convention (mostly)

https://marketplace.visualstudio.com/items?itemName=HuJohner.mc-datapack
